== Description ==

Web Video Recorder is the free version of Wordpress. Web Video Recorder offers you to share video message as post.

= Rules Document =

*   Record video using web cam and speaker.

*   Play video and watch it.

*   Upload video which show as public post.

*   You can upload video size  as per server upload limit.

== Requirements ==

1.  You need a web cam and speaker to record video.

2.  You have to increase file upload size from configuration . If you are using mysql , go to php.ini, find 'upload_max_filesize' , 'post_max_size' and change value 200mb . If you are end user query to the admin to increase file upload size.

== Installation ==

This section describes how to install the plugin and get it working.

1.  Navigate to ‘Add New’ under ‘Plugins’ menu in admin area.

2.  Upload web-video-recorder.zip from your downloaded directory or choose from WordPress Plugins repository.

3.  Activate the plugin through the ‘Plugins’ menu in WordPress.

4.  After installation , add short-code  "[rf-web-video-plugin]"  in any page where you like to show the video record section.
